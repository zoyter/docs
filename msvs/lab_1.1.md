# Лб 01. Подготовка рабочего окружения

## Задание 1. Настройте работу виртуальной машины с установленной клиентской операционной системой Windows XP (клиент) для работы в ЛВС университета.

1. Импортируйте готовую виртуальную машину с установленной ОС Windows XP (файл в формате .ova).
2. В настройках клиента перейдите в раздел "Сеть" и установите работу сетевого адаптера в режим "Сетевой моста". Также обязательно выполните генерацию нового MAC-адреса.
3. После запуска клиента установите фоновое изображение рабочего стола (изображение следует скачать из сети Интернет).
4. Измените имя компьютера в операционной системе, установив свою фамилию в качестве имени компьютера.
