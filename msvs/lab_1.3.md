# ЛР 1.2. Настройка стека протоколов TCP/IP

## Задание 1. Проверьте работоспособность стека протоколов TCP/IP.

1. Запустите виртуальную машину - клиент.
2. Откройте консоль (Пуск/Программы/Стандартные/Командная строка).
3. В командной строке введите
```bash
    ipconfig /all
```
4. Используя вывод команды из предыдущего пункта, подготовьте текстовый документ со следующими данными:

- Имя компьютера;
- Основной DNS-суффикс;
- Физический адрес;
- DHCP сервер;
- DNS-сервер;
- Аренда получена;
- Аренда истекает;
- Маска подсети;
- Шлюз по умолчанию. 

5. Убедитесь в работоспособности стека TCP/IP, отправив эхо-запросы на IP-адреса. Для этого воспользуйтесь командой ping:

- отправьте эхо-запросы на локальный адрес компьютера (loopback)
```bash
    ping 127.0.0.1 
```
(сделайте снимок экрана с результатами работы команды)

- отправьте эхо-запрос по другому IP-адресу, например 10.5.1.1  и сделайтеснимок экрана.

## Задание2. Попробуйте получить новый адрес от DHCP сервера.
1. Запустите консоль (командную строку).
2. Введите команду для сброса назначенных адресов

```
ipconfig /release
```

3. Введите команду для получения нового адреса
```
ipconfig /renew
```

4. Сделайте снимок экрана.

## Задание3. Настройте ваш виртуальный компьютер на работу с спользованием статического адреса.

1. Откройте окно  *Сетевые подключения*  (Пуск/Панель управления/Сетевыеподключения).
2. Вызовите   свойства   подключения   по   локальной   сети.   Для   этого   можновоспользоваться контекстным меню.
3. В появившемся диалоговом окне на вкладке *Общие* откройте свойства *Протокол Интернета TCP/IP*.

4. Щелкните переключатель  *Использовать*  следующий IP-адрес и введите всоответствующие поля данные:

- IP_адрес — 172.21.23**X**.**Y**, где  X –  это уникальный номер для вашейгруппы (уточните у преподавателя), а Y — это ваш номер по порядку вжурнале группы.
- Маска подсети — ```255.255.0.0```
- Основной шлюз — ```172.21.1.125```
- Предпочитаемый DNS — ```172.21.0.1```

5. Примените параметры кнопкой ОК.
6. Закройте   окно   свойств   подключения   кнопкой  *ОК*  (если   потребуется,   тосогласитесь на перезагрузку компьютера).
7. Проверьте работоспособность стека протоколов  TCP/IP  и сделайте снимокэкрана с результатами проверки.

В 5 и 7 строках находятся формулы для преобразованиядвоичных чисел в десятичную систему счисления).

1. Откройте   табличный   процессор   и   сформируйте   таблицу   по   следующемушаблону:
